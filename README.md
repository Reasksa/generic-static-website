# Nueclone — Full‑stack Next.js app

This repository contains a Next.js 14 (App Router) project that scaffolds a marketing site and a dashboard shell for a social media scheduling SaaS.

## Stack
- Next.js 14 + React 18 + TypeScript
- Tailwind CSS
- NextAuth (GitHub OAuth) + Prisma + PostgreSQL
- Stripe (webhook skeleton)

## Getting started
1. Install dependencies:
   - `npm install` or `pnpm install` or `yarn`
2. Copy env and fill values:
   - `cp .env.example .env.local`
   - Fill `DATABASE_URL`, `GITHUB_ID`, `GITHUB_SECRET`, `AUTH_SECRET`, optional `STRIPE_*`
3. Initialize database (PostgreSQL):
   - `npm run prisma:push` (creates tables)
4. Run the dev server:
   - `npm run dev`
5. Open http://localhost:3000

## Scripts
- `dev` — start Next.js in development
- `build` — build for production
- `start` — run the production build
- `lint` — lint the codebase
- `prisma:push` — sync schema to DB
- `prisma:studio` — open Prisma Studio

## Structure
- `app/` — App Router pages (marketing + dashboard), API routes
- `components/` — UI components
- `lib/` — prisma client, NextAuth, Stripe
- `prisma/schema.prisma` — data model
- `public/` — static assets
- `tailwind.config.js` — Tailwind config

## Environment variables
Copy `.env.example` to `.env.local` and set:
- `DATABASE_URL=postgresql://user:pass@host:5432/db`
- `AUTH_SECRET=` (generate: `openssl rand -base64 32`)
- `GITHUB_ID=`, `GITHUB_SECRET=` (from GitHub OAuth app)
- `STRIPE_SECRET_KEY=` (optional for later)
- `STRIPE_WEBHOOK_SECRET=` (optional for later)
